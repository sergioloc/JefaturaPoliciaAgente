# JefaturaPoliciaAgente
App de policía para primera práctica de la asignatura "Construcción y diseño de interfaces gráficas de usuario".

Perfiles de usuario:
- Policía común
- Policía con dificultades para manejar la aplicación

Análisis de tareas:
- Iniciar sesión
- Panelde emergencias
- Panel de denuncas
- Panel de eventos
- Predenuncia
- Atestado
- Sanción
- Cerrar sesión
